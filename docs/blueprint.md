# **App Name**: Nadeera's Digital Domain

## Core Features:

- Home Page: A landing page with a brief introduction and a hero image showcasing Nadeera Lakshan. Title and short bio.
- About Me: Detailed information about Nadeera, his background, and his passion for software engineering.
- Education: List of educational qualifications with institutions and dates.
- Work Experience: Details of previous work experiences, including the role and responsibilities.
- Projects: Interactive showcase of personal projects with images, descriptions, and links to GitHub or live demos.
- Skills: Visual representation of technical and soft skills using badges or progress bars.
- Certifications: A dedicated section to list certifications and achievements with issuing organizations.
- Contact Form: An interactive contact form with fields for name, email, and message, with validation and submission feedback.
- CV Generation tool: Generates a CV summary from the available info, so the user can customize it before downloading

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) for a professional and modern look.
- Background color: Light Lavender (#E6E6FA) for a soft and clean backdrop.
- Accent color: Vibrant Purple (#A020F0) for highlighting interactive elements and CTAs.
- Body text font: 'Inter', a grotesque-style sans-serif, for readability and a modern feel.
- Headline font: 'Space Grotesk' sans-serif, for headlines and a computerized, techy, scientific feel.
- Use modern, line-style icons for a clean and professional aesthetic.
- Implement a responsive grid layout that adapts to different screen sizes, ensuring a consistent user experience on all devices.
- Use subtle animations with Framer Motion to create smooth transitions and interactive elements, enhancing the overall user experience.
